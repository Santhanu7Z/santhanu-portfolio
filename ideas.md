# Design Ideas for Santhanu Ajith Kumar – AI Researcher Portfolio

## Context
Santhanu is an M.Sc. AI & ML student and researcher focused on clinical speech analysis, multimodal learning, and explainable AI. The site must feel academic-yet-modern, trustworthy, and technically sophisticated.

<response>
<idea>
**Design Movement**: Scientific Brutalism meets Editorial Minimalism

**Core Principles**:
1. Raw, high-contrast typography as the primary visual element
2. Structured asymmetry — content blocks offset on a strict invisible grid
3. Data-inspired visual motifs (waveforms, neural network nodes)
4. Monochrome palette punctuated by a single electric accent

**Color Philosophy**:
Near-black background (#0D0D0D) with off-white text and a single neon teal accent (#00E5C8). Evokes terminal aesthetics and scientific precision. The teal references signal processing and neural activation.

**Layout Paradigm**:
Left-anchored sidebar navigation (fixed), right-side scrollable content. Hero section uses oversized typographic name with a subtle animated waveform behind it.

**Signature Elements**:
1. Animated SVG waveform in the hero (references speech analysis research)
2. Monospaced code-style labels for section headings
3. Thin horizontal rule dividers with year stamps

**Interaction Philosophy**:
Hover states reveal hidden metadata (e.g., hover a project card to see tech stack). Scroll-triggered fade-ins. Cursor changes to crosshair on interactive elements.

**Animation**:
- Hero: waveform oscillates slowly on load
- Section entries: slide-up with staggered delay
- Nav: active section highlight slides smoothly

**Typography System**:
- Display: Space Grotesk (bold, 900 weight) for name/headings
- Body: IBM Plex Mono for labels, JetBrains Mono for code snippets
- Body text: Inter 400 for readable paragraphs
</idea>

<idea>
**Design Movement**: Academic Elegance — Bauhaus-inspired with warm scholarly tones

**Core Principles**:
1. Warm cream/ivory backgrounds with deep navy and terracotta accents
2. Strong typographic hierarchy using serif display + sans body
3. Generous whitespace and deliberate section rhythm
4. Subtle paper-like texture for depth

**Color Philosophy**:
Ivory (#FAF7F2) background, deep navy (#1A2744) for primary text, warm terracotta (#C4622D) as accent. Feels like a well-designed academic journal — trustworthy, intelligent, and approachable.

**Layout Paradigm**:
Single-column with wide left margins used for section labels (like academic papers). Timeline entries use a vertical rule with year stamps in the margin. Hero is typographic with a subtle geometric pattern.

**Signature Elements**:
1. Serif display font for the name (Playfair Display or Cormorant Garamond)
2. Left-margin labels in small caps
3. Thin terracotta accent lines under section headings

**Interaction Philosophy**:
Calm, deliberate transitions. Hover states use gentle color shifts. No aggressive animations — the content speaks for itself.

**Animation**:
- Fade-in on scroll with gentle upward drift
- Underline grows on hover for links
- Timeline dots pulse once on entry

**Typography System**:
- Display: Cormorant Garamond (700) for name and section titles
- Subheadings: DM Sans (600)
- Body: DM Sans (400)
- Labels/metadata: DM Sans (500) small caps
</idea>

<idea>
**Design Movement**: Dark Techno-Academic — Deep space research aesthetic

**Core Principles**:
1. Deep dark navy/slate background with luminous accents
2. Glassmorphism cards for project and publication entries
3. Subtle particle/constellation background referencing neural networks
4. Gradient accents in electric blue and violet

**Color Philosophy**:
Deep space navy (#0A0F1E) background, with electric blue (#4F8EF7) and soft violet (#A78BFA) gradients. Evokes cutting-edge AI research, deep learning, and the cosmos of data. Professional but visually striking.

**Layout Paradigm**:
Full-width sections with alternating left/right content alignment. Sticky top navigation. Hero uses large gradient text for the name with a subtle animated particle field.

**Signature Elements**:
1. Glassmorphism cards with frosted borders for projects/publications
2. Gradient text for name and key headings
3. Animated dot-grid or constellation background in hero

**Interaction Philosophy**:
Cards lift on hover with glow effects. Smooth scroll with progress indicator. Section transitions use clip-path reveals.

**Animation**:
- Hero: particles drift slowly
- Cards: lift + glow on hover
- Sections: clip-path reveal on scroll

**Typography System**:
- Display: Syne (800) for name
- Headings: Syne (700)
- Body: Nunito Sans (400/600)
